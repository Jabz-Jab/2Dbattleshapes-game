# 2Dbattleshapes-game
A fast-paced 2D action game where players control a shape and battle against waves of enemy shapes, each with unique attack patterns. Dodge, shoot, and survive intense encounters as you face powerful bosses and progressively tougher challenges.

![image alt](https://github.com/Jabz-Jab/2Dbattleshapes-game/blob/055746f14d6de7bc2b639010ac03549e822a3903/Screenshot%202026-04-01%20064954.png)

## 🚀 How to Run

Make sure you have Python installed (recommended: Python 3.10+)

### 1. Install dependencies
```bash
pip install pygame
```
### 2. Run to this file
```bash
python geometry_duel/main.py
```
